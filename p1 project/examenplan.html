Examenplan Software Developer MBO 4

Een korte samenvatting van het examenplan voor leerjaar 3 van Software Developer MBO 4.

Beroepsspecifieke onderdelen

Hier onder staan kort uitgelegd de onderdelen die een leerling moet kunnen om hun diploma te halen in leerjaar 3

  • Software kunnen maken
  • Voortgang en werkzaamheden van een project kunnen bewaken
  • Software kunnen ontwerpen
  • Kunnen werken aan een specifiek onderdeel van een project en dit kunnen integreren met de rest.
  • Software testen
  • Verbetervoorstellen voor software kunnen maken
  • Werken in een team
  • Overleg kunnen voeren
  • Werk klunnen presenteren
  • Reflecteren op verrichte werk

Dit is allemaal heel simpel neergezet. In de loop van de opleidng zullen wij dieper op deze onderwerpen ingaan.
